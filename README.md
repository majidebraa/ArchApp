# Base Project for Android Kotlin Applications
## Architecture Components
This application implements the following concepts:

  * MVVM + Data binding
  * Multi-modules
  * Navigation Component
  * ViewModel
  * LiveData
  * Coroutines
  * Koin
  * MockWebServer
  * Mockk
